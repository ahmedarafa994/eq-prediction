### 🛠️ The "CNN 2.0" Architectural Upgrade

To break the 0.10 barrier, we need to widen the "keyhole" through which the model sees the audio. We are going to double the filter depth and add a bit more complexity to the feature extraction.

#### The New Filter Stack
Instead of starting with 16 filters, we start with **32** and scale up to **256**. This allows the model to detect 256 unique "spectral fingerprints" simultaneously.

| Layer | Old Filters | **New Filters** | **Why?** |
| :--- | :--- | :--- | :--- |
| **Conv 1** | 16 | **32** | Captures finer initial spectral deltas. |
| **Conv 2** | 32 | **64** | Defines basic shapes (Bell curves vs. High-pass). |
| **Conv 3** | 64 | **128** | Identifies Q-factor interactions. |
| **Conv 4** | 128 | **256** | High-level abstraction of the 8-band relationship. |
